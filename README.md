# React Todo List

React todo list for interview testing

# Features
- Add task
- Edit task
- Delete task
- Save task on localStorage

# Installation
1. Using npm to install package
```bash
npm install
```

# How to run
In root of project run this in terminal for testing in dev mode
```bash
npm run dev
```
